# Diabetes Prediction using Machine Learning

## 📌 Project Overview
This project predicts whether a person is diabetic or not using Machine Learning.
The model is trained using Logistic Regression.

## 📊 Dataset
- Pima Indians Diabetes Dataset
- Features include glucose, blood pressure, BMI, age, etc.

## ⚙️ Technologies Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## 🚀 Steps Performed
1. Data Loading
2. Exploratory Data Analysis (EDA)
3. Data Visualization (Heatmap)
4. Train-Test Split
5. Model Training (Logistic Regression)
6. Model Evaluation (Accuracy, Confusion Matrix)
7. Model Saving using Joblib
